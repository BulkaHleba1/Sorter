# Sorter
Небольшая консольная утилита, которая сортирует папку "Downloads".
В папку Images идут файлы-изображения, в Texts идут текстовые файлы, включая файлы .md, в папку Videos идут файлы-видео. В папку Archuves уйдут файлы-архивы, zip, rar, tar.gz...
Прошу заметить, после запуска указанной папке в конфиге (редактируйте sorter_config.json), все файлы будут рассортированы. Файлы .py, т.п. будут отправлены в папку unknown.
